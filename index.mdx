## What is D&D Textil?

**D&D Textil** is a full-stack e-commerce platform built for a digital textile store. It features a React 18 frontend with Vite, a Java backend for PostgreSQL connectivity, and a rich set of role-based dashboards for administrators, sellers, and clients.

<Info>
  This project was developed as part of the SENA ADSO program (Análisis y Desarrollo de Software).
</Info>

## Tech stack

| Layer | Technology |
|---|---|
| Frontend | React 18, Vite, Tailwind CSS |
| Animation | Framer Motion |
| Forms | React Hook Form + Zod |
| Routing | React Router v6 |
| Charts | Recharts |
| Backend | Java (PostgreSQL driver) |
| Database | PostgreSQL |

## User roles

D&D Textil supports three distinct user roles, each with dedicated dashboards and route-level protection:

| Role | Access |
|---|---|
| **Client** | Browse catalog, manage cart, checkout, view order history, wishlist, support tickets |
| **Seller** | Manage products, view orders, monitor stock alerts |
| **Admin** | Full access: users, inventory, analytics, vendor moderation, system configuration |

<Tip>
  See [Authentication](/features/authentication) for role setup and protected route configuration.
</Tip>
